# anime-list-and-note-list-with-cookie-authentication
Backend for anime list and note list  app with anime search using external graphql api and cookie based user authentication.

Backend uses 
              <ul><li>NodeJS</li>
              <li>expressJS</li>
              <li>passportJS</li>
              <li>mongoDB</li>
              <li>mongoose</li>
              <li>express-session</li></ul>
              
Frontend uses
              React
              tailwind
